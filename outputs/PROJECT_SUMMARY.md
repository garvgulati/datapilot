# DataPilot Deliverables

The workspace contains a complete full-stack SaaS codebase with:

- Next.js, TypeScript, Tailwind, and Recharts frontend.
- FastAPI, Pandas, SQLAlchemy, PostgreSQL, and OpenAI-ready backend.
- Docker Compose setup.
- SAP-style sample project-cost dataset.
- API documentation, architecture notes, screenshot placeholders, resume bullets, and demo script.

Start with:

```bash
cp .env.example .env
docker compose up --build
```

Then open:

- Frontend: `http://localhost:3000`
- Backend docs: `http://localhost:8000/docs`
