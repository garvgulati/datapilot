# Resume Bullet Points

- Built a full-stack SaaS analytics platform using Next.js, TypeScript, FastAPI, PostgreSQL, Pandas, SQLAlchemy, Docker, and OpenAI APIs to automate business dataset ingestion, cleaning, analysis, visualization, and natural-language Q&A.
- Implemented CSV/Excel upload workflows with dataset profiling, duplicate detection, missing-value handling, column standardization, type inference, and persistent cleaned dataset storage.
- Developed a reusable analytics engine that generates KPI cards, descriptive statistics, trend summaries, numerical outlier detection, chart-ready JSON, and table previews from arbitrary business datasets.
- Created a grounded AI data analyst chat that constrains responses to uploaded dataset context and returns auditable Pandas logic for questions such as cost overruns, budget variance, and executive summaries.
- Designed a SAP-style Project Cost Intelligence dashboard with planned vs actual cost analysis, WBS-level breakdowns, over-budget alerts, variance rankings, and executive summary generation.
- Containerized the frontend, backend, and PostgreSQL database with Docker Compose and documented local setup, API routes, architecture, seed data, and demo flow for production-style handoff.
